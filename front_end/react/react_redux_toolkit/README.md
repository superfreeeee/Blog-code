# React + TS 前端应用

# Get Started 项目启动

```bash
# instal dependency if cli action fail
yarn install

# start application in development mode
yarn start
```

# 技术选型

- 核心技术栈：`React`
  - 状态管理：`redux`
  - 路由：`react-router-dom`
  - CSS 支持：`styled-component、scss`、CSS Module
  - API 请求：axios
- 类型语法：`typescript`
- 工程化：`babel + webpack`
